# ChapterListView

- **Type:** Structure
- **Module:** Strobe
- **DocC Path:** `/documentation/strobe/chapterlistview`

Displays the chapter list for a document with progress indicators.

## Overview

Shows a “Read Full Document” option and individual chapters with not-started / in-progress / completed status based on the current reading position.

## API

### Initializers
- `init(document: Document)`

### Instance Properties
- `var body: some View`
- `var document: Document`
