# ContentView

- **Type:** Structure
- **Module:** Strobe
- **DocC Path:** `/documentation/strobe/contentview`

The main library view displaying imported documents in a grid.

## Overview

Handles document import (PDF/EPUB via the system file picker), legacy word storage migration, and navigation to the reader or chapter list.
